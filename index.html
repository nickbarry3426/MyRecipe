Google Drive Cloud Sync

Sync your recipes automatically between your iPad and Mac using a Google Apps Script.

How to set up:

  1. Copy the embedded Google Apps Script below.
  2. Go to script.google.com and create a new project.
  3. Paste the script code, save, and click Deploy > New Deployment.
  4. Set Select type to Web App, Execute as Me, and Who has access to Anyone.
  5. Deploy, authorize Google Drive permissions, copy the Web App URL and paste it above.
